# Whisper Flow Documentation

Whisper Flow is an open-source local voice dictation tool with AI-powered text editing. It's a privacy-focused alternative to commercial voice dictation apps like Wispr Flow.

## Key Features

- **100% Local Transcription** - Uses faster-whisper for offline speech-to-text
- **AI Text Editing** - Clean up transcriptions with Ollama, OpenAI, or Anthropic
- **Personal Dictionary** - Auto-learn names, technical terms, and custom vocabulary
- **Snippets** - Text expansion shortcuts (say "sig" â†’ get your full signature)
- **Voice Commands** - "Delete that", "new line", "undo" and more
- **App Context Awareness** - Auto-adjusts tone based on active application (macOS)
- **Cross-Platform** - Works on macOS and Windows with floating overlay UI
